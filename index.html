Chapter 1 : Technology

To understand the technology of a firm, the production function is the easiest way to summarize it. General and useful ways to understand technology involve understanding firms’ production possibilities.

Measurement of Inputs and Outputs

A firm produces output from various combinations of inputs. Suppose:

  • (q)( q )(q) is the output,
  • (L)( L )(L) is labor,
  • (K)( K )(K) is capital.

For specific combinations:

  • (q=f(1,2))( q = f(1, 2) )(q=f(1,2))
  • (q=f(2,2))( q = f(2, 2) )(q=f(2,2))
  • (q=f(2,1))( q = f(2, 1) )(q=f(2,1))

This representation shows the combinations of inputs and output that the firm is able to produce.

General Argument:
To study firm choices, we need a structured and convenient way to summarize the production possibilities of the firm.

What are Production Possibilities?

Production possibilities refer to the combinations of inputs and outputs that are technologically feasible for a firm. This concept is crucial in understanding how firms utilize their resources efficiently to produce goods and services.

Key Points:

  1. Inputs and Outputs:

    • In production, both inputs (like labor and capital) and outputs (the goods produced) flow over a specific period.
    • For example, a certain amount of inputs are used to produce a specific quantity of output during that period.

  2. Time Dimension:

    • When specifying inputs, it is essential to include a time dimension.
    • For instance, labor can be measured in hours worked per week, while capital can be assessed based on the amount spent on machinery per month.
    • Important Note: It’s crucial to use the same time unit for analysis to ensure consistency.

  3. Spatial and Temporal Nature of Production:

    • Some aspects of production involve spatial and temporal considerations.
    • For example, concrete that is available in a given year may be used to construct a building in the following year.
    • Similarly, concrete used in your city might have been sourced from a neighboring state. Therefore, the input “concrete” must be understood as available in a specific location at a particular time.

Takeaway:

All these points are vital when analyzing producers’ behavior and focusing on inputs and outputs. However, it is common for us to overlook these aspects in formal discussions. This oversight does not imply that these factors are unimportant; rather, it is often assumed that they are adequately accounted for in the analysis.

Specification of Technology

The specification of technology provides a general framework that captures the same intuition but offers a different formalization for studying production behavior.

General Framework:

  • Consider a firm that has nnn possible goods.
  • This framework is general because it allows all nnn possible goods to serve as inputs and/or outputs.

Inputs and Outputs:

  • It’s easy to think of examples where certain goods act purely as inputs or outputs.
  • Examples:
    • In chemical production, heat can serve as both an input (for energy) and an output (as a byproduct).

Notation:

  • Let yjiy_j^iyji​ represent the units of good jjj used as an input.
  • Let yjoy_j^oyjo​ represent the units of good jjj produced as an output.
  • The net output of good jjj is given by:
    yj=yjo−yji y_j = y_j^o - y_j^i yj​=yjo​−yji​
  • If yjo>yjiy_j^o > y_j^iyjo​>yji​, the firm is producing more of good jjj than it is using.
  • Conversely, if yjo<yjiy_j^o < y_j^iyjo​<yji​, the firm is using more of good jjj as an input (e.g., curd) and producing less of it (e.g., heat).

Production Plan:

A production plan lists the net outputs of various goods. It can be represented as a vector:
y=(y1,y2,…,ym) y = (y_1, y_2, \ldots, y_m) y=(y1​,y2​,…,ym​)
where yyy is in Rn\mathbb{R}^nRn.

Production Possibility Set (PPS):

The set of all technologically feasible production plans is called the Production Possibility Set (PPS), denoted as YYY.

  • YYY is obviously a subset of Rn\mathbb{R}^nRn and describes all patterns of inputs and outputs that are technologically feasible, providing a complete description of the firm’s technological possibilities.

Examining Technological Possibility Set:

We may want to distinguish between production plans that are “immediately” feasible and those that are “eventually” feasible. This distinction aligns with the familiar discussion of the short run (SR) and the long run (LR).

Short Run vs. Long Run:

  • In the short run, since some factors are fixed, production plans must take these restrictions into account.
  • We will assume that all restrictions faced by the firm in the short run are described by a vector zzz in Rn\mathbb{R}^nRn.
    • Think of zzz as the list of the maximum amounts of various inputs and outputs that can be produced in the concerned period.

Restricted Production Possibility Set:

We use a restricted production possibility set denoted by Y(z)Y(z)Y(z) (in the short run, this is called the SR PPS).
Y(z)={y∈Y:Yj=yj} Y(z) = \{ y \in Y : Y_j = y_j \} Y(z)={y∈Y:Yj​=yj​}

  • Here, Y(z)Y(z)Y(z) consists of all feasible net output bundles consistent with the set of restrictions.
  • Example: If a factor is fixed at yny_nyn​ in the SR, then the SR PPS can be defined as:
    Y(yˉn)={y∈Y:yn=yˉn} Y(\bar y_n ) = \{ y \in Y : y_n =\bar y_n \} Y(yˉ​n​)={y∈Y:yn​=yˉ​n​}

Conclusion:

The set Y(z)Y(z)Y(z) is a subset of YYY because it contains all production plans that satisfy the additional conditions imposed by the input requirements.

Input Requirement Set

Suppose a firm produces a single output, denoted as yyy, while the rest of the goods are inputs. We can denote the vector of inputs as x=(x1,x2,…,xn)x = (x_1, x_2, \ldots, x_n)x=(x1​,x2​,…,xn​). The production process can be expressed as the net output bundle (y,−x)(y, - x)(y,−x), where yyy represents the units of output produced.

A special case of the restricted production possibilities set is the input requirement set, denoted as V(y)V(y)V(y). It is defined as:
V(y)={x∈R+n:(y,−x)∈Y} V(y) = \{ x \in \mathbb{R}^n_+ : (y, -x) \in Y \} V(y)={x∈R+n​:(y,−x)∈Y}
This set describes the real space of nnn dimensions in the positive space.

Properties of V(y)V(y)V(y):

  • V(y)V(y)V(y) has two parts:
    1. All inputs are constrained in the positive real space.
    2. The net output bundle associated with these inputs is technologically feasible.

Thus, V(y)V(y)V(y) is the set of all input bundles that can produce at least yyy units of output.

Important Note:

The input requirement set defines inputs as positive numbers, which is the opposite of how inputs are defined in the Production Possibility Set (PPS).

What do we mean by “efficient”?

A production plan yyy is said to be efficient if there is no other production plan y′y'y′ such that y≤y′y \leq y'y≤y′ and y′≠yy' \neq yy′=y. In simple terms, this means that a production plan is efficient if there is no way to produce more output with the same inputs or produce the same output with fewer inputs.

Activity Analysis

This focuses on how to describe production processes. The simplest way to describe production using input requirement sets is to simply list them!

Example:

Suppose we have an output good, produced using two factor inputs: factor 1 and factor 2. There are two different activities or techniques by which this production takes place. To produce 1 unit of the output good, we can do either of the following:

  1. Technique 1: 1 unit of factor 1 and 2 units of factor 2 →\rightarrow→ 1 unit of output.
  2. Technique 2: 2 units of factor 1 and 1 unit of factor 2 →\rightarrow→ 1 unit of output.

Now, let’s further define the goods:

  • Suppose output is good 1.
  • Input 1 is good 2.
  • Input 2 is good 3.

Production Possibility Set:

Y={(1,−2,−1),(1,−1,−2)} Y = \{(1, -2, -1), (1, -1, -2)\} Y={(1,−2,−1),(1,−1,−2)}

Input Requirement Set:

For 1 unit of output, the input requirement set is:
V(1)={(2,1),(1,2)} V(1) = \{(2, 1), (1, 2)\} V(1)={(2,1),(1,2)}

Now, if I ask you, what is V(y)V(y)V(y)? That is, what is the set of feasible ways to produce yyy units of output?

V(y)={(y,2y),(2y,y)} V(y) = \{(y, 2y), (2y, y)\} V(y)={(y,2y),(2y,y)}

Does this set include all relevant possibilities? Not quite! Why?

  • It is true that (y,2y)(y, 2y)(y,2y) produces yyy units if we use technique A.
  • Similarly, (2y,y)(2y, y)(2y,y) produces yyy units if we use technique B.

But what if we use a mixture of techniques A and B?

Let us try to formulate a more accurate input requirement set for y=1,2,…y = 1, 2, \dotsy=1,2,…. Let:

  • YAY_AYA​ be the amount of output produced by technique A.

Monotonic Technologies:

Now, let’s focus on some properties of this input requirement set.

In this two-technique example, suppose we have an input vector of (3,2)(3, 2)(3,2). Then, if I ask you: Is this sufficient to produce 1 unit of output?

Your answer should be Yes! Why?

Because either technique 1 or technique 2 will allow us to produce 1 unit of output. However, we would need to dispose of the extra inputs along the way.

If such free disposal is allowed, it is reasonable to argue that:

  • If xxx is a feasible vector of inputs that produces yyy units of output,
  • Then, any input vector x′x'x′, with at least as much of each input as xxx, should also be feasible to produce yyy.

Thus, the input requirement set is monotonic:
If x∈V(y) and x′≥x, then x′∈V(y). \text{If } x \in V(y) \text{ and } x' \geq x \text{, then } x' \in V(y). If x∈V(y) and x′≥x, then x′∈V(y).

Similarly, for production sets:

  • If y∈Yy \in Yy∈Y and y′≤yy' \leq yy′≤y, then y′y'y′ is in YYY.
  • This means that production plan y′y'y′ produces equal or smaller amounts of output by using at least as much inputs as yyy.

Naturally, if yyy is feasible, then y′y'y′ is also feasible.

Convex Technologies

Now, using the same two techniques, suppose we want to produce 100 units of output. The question arises:

What will the input requirement set look like?

The straightforward answer: We could replicate the process and thereby produce 100 times as much.

Is this realistic? Not always!

Suppose it is realistic, then using V(1)={(1,2),(2,1)}V(1) = \{(1, 2), (2, 1)\}V(1)={(1,2),(2,1)}, we can replicate it for y=100y = 100y=100. This would give us the input combinations:
V(100)={(100,200),(200,100)} V(100) = \{(100, 200), (200, 100)\} V(100)={(100,200),(200,100)}

Any other possible ways to produce 100 units of output?

Consider the following:

  • 50 units of activity A and 50 units of activity B.

  • 50 units of activity A uses the combination (100,50)(100, 50)(100,50).

  • 50 units of activity B uses the combination (50,100)(50, 100)(50,100).

Hence, these combinations would use 150 units of input 1 and 150 units of input 2. So, (150,150)(150, 150)(150,150) should also be in V(100)V(100)V(100).

Another scenario:

Suppose we want to use the following combination:

  • 25 processes of activity A: (50,25)(50, 25)(50,25).
  • 75 processes of activity B: (25,150)(25, 150)(25,150).

Thus, the input combination (125,175)(125, 175)(125,175) should also be in V(100)V(100)V(100).

Generalization:

Let’s make this more general. If we use a weighted combination of the two techniques, we get:
t(100,200)+(1−t)(200,100)=(100t+200(1−t),200t+100(1−t)) t(100, 200) + (1-t)(200, 100) = (100t + 200(1-t), 200t + 100(1-t)) t(100,200)+(1−t)(200,100)=(100t+200(1−t),200t+100(1−t))

This property is called Convexity.

Convexity Property:


For the input requirement set V(y)V(y)V(y), if xxx and x′x'x′ are in V(y)V(y)V(y), then any convex combination of xxx and x′x'x′ is also in V(y)V(y)V(y). Mathematically:
tx+(1−t)x′∈V(y)for all 0≤t≤1 tx + (1-t)x' \in V(y) \quad \text{for all } 0 \leq t \leq 1 tx+(1−t)x′∈V(y)for all 0≤t≤1
Thus, V(y)V(y)V(y) is a convex set.

Is convexity a reasonable assumption?

When scaling up from small to large outputs, convexity is a reasonable assumption. However, this can be questionable because, at times, the scale of the underlying activities may become too large.

Convexity may not be reasonable under certain assumptions but may be reasonable in other circumstances. For example:

  • Suppose we measure output per month.
  • If activity A produces yyy units of output in one month, and activity B also produces yyy units in one month, then we can split the processes.
  • We can use half of activity A for the first half of the month and half of activity B for the second half, producing yyy units of output.

Convexity Assumption for Production Set:

If yyy is in YYY and y′y'y′ is in YYY, then the convex combination:
ty+(1−t)y′∈Yfor all 0≤t≤1 ty + (1-t)y' \in Y \quad \text{for all } 0 \leq t \leq 1 ty+(1−t)y′∈Yfor all 0≤t≤1
implies that YYY is a convex set.

A Simple Property:

One simple property of a convex production set is that it implies a convex input requirement set as well.
